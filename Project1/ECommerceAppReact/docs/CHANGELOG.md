## [1.1.0] - 2026-01-12

### Added
- ✅ **Enhanced Search & Filter** for products:
  - Search by product name or description
  - Filter by category
  - Filter by price range (min/max)
  - Sorting options (newest, price, name)
  - Combined filters support
  - Active filter badges with clear all button
  - Empty state UI when no products found

---

## [1.0.0] - 2026-01-12

### Added
- ✅ Initial project setup dengan Clean Architecture
- ✅ User authentication (Login/Register) dengan ASP.NET Core Identity
- ✅ Product catalog dengan kategori
- ✅ Shopping cart functionality
- ✅ Order management system
- ✅ Seller dashboard untuk manage products
- ✅ Content Manager area untuk Announcements
- ✅ Multi-language support (Indonesian/English)
- ✅ Product seeder dengan 100+ dummy products
- ✅ Premium UI design dengan modern styling

### Technical
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server database
- Clean Architecture (Domain, Application, Infrastructure, Web)
